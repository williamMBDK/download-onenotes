Tangentens ligning og bevis

Vi vil kigge på tangentens ligning (y).

y − y 0

Bevis:

y = f ′ x 0 ⋅ x − x 0 + f ( x 0 )

y   −   y 0 =   𝛼 x − x 0 → y =   𝛼 x − x 0 +   y 0

Vi vil nu angive 𝛼   o g   y 0 .

y 0 = f x 0 𝛼 = f ′ x 0

Indsættes I ligningen:

y = f ′ x 0 x − x 0 + f x 0

Sætning 9.6:

En differentiabel funktion (f(x)) har I punkte-t ( x 0 ,   f ( x 0 ) )   en tangent til grafen givet ved:

y = f ′ x 0 x − x 0 + f x 0
