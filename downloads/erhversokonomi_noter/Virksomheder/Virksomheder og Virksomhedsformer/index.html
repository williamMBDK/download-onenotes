Virksomheder og Virksomhedsformer

Enkeltmandsejede virksomheder:

  • Ejerforhold: En ejer

    • Sørger for forpligtelser
  • Økonomi: sammen med privatøkonomi
  • Gæld: personligt ansvar

  • Beskatning:

    • Personskattereglerne
    • Virksomhedsordningen
    • Kapitalafkastordningen
  • Krav om startkapital: ingen
  • Krav om regnskab: skal ikke offentliggøres, men skal bogføres. Ingen årsrapport.
  • Selvansvar
  • Privatøkonomi og virksomhed I et / flettet sammen.

  • Hæfter personligt (gæld, konkurs)

    • Kan miste alt



Interessentselskaber I/S:

  • Ligesom enkeltmandsejede
  • Flere ejere
  • Valg af leder
  • Hæfter Personligt og Solidarisk
  • kontrakt

Anpartsselskaber Aps:

  • Kun ansvar for startkapital.
  • Beskatning I virksomhed og ikke ejerne I henhold til selskabskattereglerne.

  • Krav:

    • Stiftelsesdokument
    • Direktion + bestyrelse eller sikkerhedsråd
    • Årsrapport skal indsendes.
  • Startkapital: 80k
  • 670 - 2150 at opstarte virksomhed.


Andelsselskab:

  • Personligt og solidarisk ansvar.
  • Nye medlemmer -> generelforsamling.

Aktieselskaber (a/s):

  • 500k for opstart --> aktiekapitalens størrelse

  • Ejes af aktionærer.

    • A og B aktier evt. (procent afgør indflydelse)
    • Ansætter bestyrelse (typisk tidlgere direktører og aktionærer) + bestyrelsesformand (ikke direktøren)
  • Kan aldrig miste mere end den kapital der er indskudt.

  • Direktion (daglig drift) og bestyrelse

    • Bestyrelse ansætter direktion og direktør.
  • Generelforsamling: aktionære kan komme med forslag og stemme om tiltag.
  • Årsrapport skal godkendes.
  • 670 - 2150 at opstarte virksomhed

  • Hvis man går konkurs går gælden tabt. Dvs man skal ikke betale det.

    • Man kan starte ny virksomhed med det samme.
    • Retsager
    • Kurator sættes på for at finde ud af hvad virksomheden afgører.

  • Ulempe:

    • kan ikke eje 100% og selv være direktør hvis man selv er bestyrelsesformand.
    • Offenliggøre flere tiing
    • Afgiver indflydelse

  • Fordele

    • Kommer hurtig I vækst
    • Sikkerhed grundet ansvaret ved gæld..
